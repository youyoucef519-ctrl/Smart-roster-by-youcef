⚙️ Setup Required

To make this roster work online for all staff, you need to connect it to a free Supabase database. Follow these steps:

1
Go to supabase.com and create a free account, then create a new project called bakery-roster
2
Go to Settings → API and copy your Project URL and anon public key
3
Go to SQL Editor in Supabase and run the setup SQL that came with this file
4
Open this HTML file in a text editor, find SUPABASE_URL and SUPABASE_KEY near the top of the script and replace with your values
